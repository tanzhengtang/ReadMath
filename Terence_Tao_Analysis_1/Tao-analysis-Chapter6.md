## 6.1 Convergence and limit laws。
### Proposition 6.1.12  
* If $a_n$ is a convergent sequence, then we have $\forall ε/2 > 0 \  \exist n,n^o \geq M$ such that $|a_n - L| < ε/2$ and $|a_{n^o} - L| < ε/2$. $|a_n - a_{n^o}| = |a_n - L + L - a_{n^o}| \le |a_n - L| + |a_{n^o} - L| < ε/2 + ε/2 < ε$. Then $a_n$ is a Cauchy seqencue.
